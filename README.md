```bash
# Navigate to project directory

# Run development server
npm run dev
```

## Usage

1. Click "Start New Workout" to create a new workout
2. Add exercises and fill in sets, reps, and weight
3. Click "Save" to save your workout
4. View past workouts on the home screen
5. Right-click or long-press to delete workouts
